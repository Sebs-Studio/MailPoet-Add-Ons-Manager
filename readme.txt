== Description ==

Allows you to find, install and activate add-ons created for MailPoet and third-parties with a few clicks of a button.

This plugin will be added to the core of the MailPoet plugin in future. You may use this code to create your own add-on manager.

= Contributing =

If you have an add-on or service for MailPoet that you would like to be listed, then please contact MailPoet (http://www.mailpoet.com/contact/)

== Installation ==

= Manual installation =

The manual installation method involves downloading my plugin and uploading it to your webserver via your favourite FTP application.

1. Download the plugin file to your computer and unzip it
2. Using an FTP program, or your hosting control panel, upload the unzipped plugin folder to your WordPress installation's wp-content/plugins/ directory.
3. Activate the plugin from the Plugins menu within the WordPress admin.
